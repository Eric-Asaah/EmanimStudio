EMANIM STUDIO - MATHEMATICAL ANIMATION STUDIO
=============================================

QUICK START:
✅ Double-click "EmanimStudio.exe"
✅ No installation required!
✅ Choose [1] for Terminal Interface
✅ Browse 65+ mathematical animations
✅ Render professional videos instantly!

SYSTEM REQUIREMENTS:
• Windows 7/8/10/11
• 4GB RAM (8GB recommended for 4K)
• 2GB free space

FOLDER STRUCTURE:
• EmanimStudio.exe - Main launcher
• emanim_terminal.bat - Animation creator
• Animations/ - 65+ pre-built templates
• Output/ - Your rendered videos go here
• Lib/ - Contains all dependencies (Python, Manim, FFmpeg)

GETTING STARTED:
1. Launch EmanimStudio.exe
2. Select [1] for Terminal Interface
3. Choose animation category
4. Select animation template
5. Set quality (480p to 4K)
6. Render and enjoy!

TROUBLESHOOTING:
• Run as Administrator if you have permission issues
• Ensure all files are in the same folder
